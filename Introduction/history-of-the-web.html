A Tale Of Web

by: Babajide June 18th, 2023. 12:38AM

Once upon a time, the web didn't exist. At least not the way we know it now. However some really smart guy at some research lab thought of way for researchers, scientists, engineers and professors to share information with each other accross geographical by linking to and from diffrent sources of information stored on difrent computers accross the globe.

The name of that smart guy is Lee Berners, a british computer scientist. He invented the World Wide Web in 1989. By October 1990 Tim had written the fundamental technologies that remain the foundation of today. The technologies are;

  1. HTML - HyperText Markup Language
  2. HTTP - HyperText Transmission Protocol
  3. URI - Uniform Resource Identifier
  4. HTTPD - HyperText Transmission Protocol Daemon.
  5. Browser - World Wide Web Application for rendering and viewing HTML He launched the first webpage in 1990.

Today HTML is the foundational stucture of the web.

How does HTML fit in with the two web other technologies?

HTML provides the structure of the webpage

CSS - Cascading Style Sheets, provides the presenation of the webpage, how it looks and feels

Javascript - is a programing language that provides the functionality, interactivity and behaviour of a webpage.

This course teaches the fundamentals of web developemnt and explores how these technologies can be used in combination to build simple and advanced Applications, User Interfaces, and Components that run in the Browser, on any device that has access to the internet.

Note: The world web web isn't the Internet, it rides on the internet, that is, it uses the internet as its underlying technology to access, transport and connect to diffrent computers globally that send and recieve web resources.
